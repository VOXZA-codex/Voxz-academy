📘 PEMBELAJARAN LENGKAP HTML

Apa Itu HTML?

HTML adalah singkatan dari HyperText Markup Language. Ini adalah bahasa standar yang digunakan untuk membuat struktur dan kerangka dasar sebuah halaman situs web. HTML bukanlah bahasa pemrograman, melainkan bahasa penanda yang berfungsi mengatur bagaimana teks, gambar, tautan, tabel, dan elemen lain ditampilkan di peramban (browser).

Setiap halaman web yang kamu buka di internet pasti dibangun menggunakan HTML sebagai dasarnya. HTML bekerja dengan cara memberikan "tanda" atau penanda pada konten agar browser mengerti apakah itu judul, paragraf, gambar, atau bagian lainnya.

Sejarah Singkat HTML

HTML pertama kali dikembangkan oleh Tim Berners-Lee pada tahun 1991. Seiring perkembangan teknologi internet, HTML terus mengalami pembaruan:

  • HTML 1.0: Versi pertama yang sangat sederhana.
  • HTML 2.0: Menjadi standar resmi pada tahun 1995.
  • HTML 4.01: Versi yang banyak digunakan selama bertahun-tahun.
  • HTML5: Versi terbaru dan paling modern yang dirilis secara resmi pada tahun 2014. Versi ini mendukung fitur multimedia, grafis, dan struktur yang lebih baik tanpa memerlukan plugin tambahan.

Struktur Dasar Dokumen HTML

Setiap berkas HTML memiliki struktur dasar yang wajib ada agar bisa dibaca dengan benar oleh browser. Berikut bentuk strukturnya:

<!DOCTYPE html>
<html lang="id">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Judul Halaman</title>
</head>

<body>
  <!-- Semua konten terlihat ada di sini -->
</body>

</html>

Penjelasan Setiap Bagian:

  • <!DOCTYPE html>: Memberitahu browser bahwa dokumen ini menggunakan standar HTML5.
  • <html>: Tag induk yang membungkus seluruh isi dokumen HTML.
  • <head>: Bagian yang berisi informasi pengaturan halaman, seperti judul, pengkodean karakter, dan tautan ke berkas lain.
  • <title>: Menentukan teks yang muncul di bilah judul jendela browser.
  • <body>: Tempat semua konten yang akan terlihat oleh pengunjung diletakkan, seperti teks, gambar, tombol, dan tabel.

Mengenal Tag, Elemen, dan Atribut

Dalam menulis kode HTML, ada 3 istilah utama yang perlu dipahami:

1. Tag

Tag adalah penanda awal dan akhir dari sebuah elemen. Biasanya ditulis dengan tanda kurung siku < >. Kebanyakan tag memiliki pasangan, yaitu tag pembuka dan tag penutup yang diberi tanda garis miring /.

Contoh: <p> (pembuka) dan </p> (penutup)

2. Elemen

Elemen adalah keseluruhan bagian dari tag pembuka, isi konten, hingga tag penutupnya.

<p>Ini adalah sebuah paragraf dalam elemen HTML</p>

3. Atribut

Atribut adalah informasi tambahan yang diberikan pada sebuah elemen untuk mengatur sifat atau perilakunya. Atribut ditulis di dalam tag pembuka dan memiliki format nama="nilai".

<a href="halaman.html" target="_blank">Klik di sini</a>

Pada contoh di atas, href dan target adalah atribut.

Tag-tag Umum yang Sering Digunakan

🔹 Tag Judul

HTML menyediakan 6 tingkat judul, dari yang paling besar hingga paling kecil:

<h1>Judul Utama</h1>
<h2>Judul Bagian</h2>
<h3>Sub Judul</h3>
<h4>Judul Kecil</h4>
<h5>Judul Lebih Kecil</h5>
<h6>Judul Paling Kecil</h6>

🔹 Tag Teks dan Paragraf

  • <p> → Membuat paragraf teks baru.
  • <b> atau <strong> → Membuat teks tebal.
  • <i> atau <em> → Membuat teks miring.
  • <u> → Memberi garis bawah pada teks.
  • <br> → Memindahkan teks ke baris baru (tag tunggal, tidak perlu penutup).
  • <hr> → Membuat garis pemisah antar bagian konten.

🔹 Tag Gambar dan Tautan

Untuk menampilkan gambar:

<img src="nama-gambar.jpg" alt="Keterangan gambar" width="300">

Untuk membuat tautan ke halaman lain:

<a href="https://voxz.my.id" target="_blank">Kunjungi Situs Saya</a>

🔹 Tag Daftar

Daftar tidak berurutan (menggunakan titik):

<ul>
  <li>Item Pertama</li>
  <li>Item Kedua</li>
</ul>

Daftar berurutan (menggunakan nomor):

<ol>
  <li>Langkah Satu</li>
  <li>Langkah Dua</li>
</ol>

HTML5: Fitur dan Keunggulan

HTML5 membawa banyak perubahan dan kemudahan dibanding versi sebelumnya. Beberapa keunggulannya antara lain:

  • Struktur Lebih Jelas: Memiliki tag khusus seperti <header>, <nav>, <main>, <section>, <article>, dan <footer> untuk memudahkan pembagian bagian halaman.
  • Dukungan Multimedia: Bisa langsung menampilkan video dan audio tanpa bantuan program tambahan menggunakan tag <video> dan <audio>.
  • Grafis Interaktif: Memiliki elemen <canvas> dan <svg> untuk membuat gambar dan animasi langsung di halaman.
  • Kompatibilitas Tinggi: Berjalan lancar di berbagai jenis perangkat, mulai dari komputer, tablet, hingga ponsel pintar.
💡 Catatan Penting: Ingatlah bahwa HTML hanya mengatur struktur. Untuk mengatur tampilan seperti warna, ukuran, tata letak, dan gaya desain, kamu akan menggunakan CSS. Sedangkan untuk membuat halaman bisa berinteraksi dan bergerak, kamu akan mempelajari JavaScript. Ketiganya bekerja sama untuk membuat situs web yang bagus dan fungsional.

Kesimpulan

Menguasai HTML adalah langkah pertama dan paling mendasar bagi siapa saja yang ingin belajar membuat situs web. Semakin kamu memahami cara kerja tag, elemen, dan strukturnya, semakin mudah pula nanti saat mempelajari materi selanjutnya. Latihanlah dengan menulis kode secara langsung, karena cara terbaik menguasai HTML adalah dengan terus mempraktikkannya.
