⚡ PEMBELAJARAN LENGKAP JAVASCRIPT

Apa Itu JavaScript?

JavaScript atau sering disingkat JS adalah bahasa pemrograman tingkat tinggi yang berfungsi membuat halaman web menjadi interaktif, dinamis, dan bisa merespons tindakan pengguna. Kalau HTML adalah kerangka dan CSS adalah tampilannya, maka JavaScript adalah "otak" yang mengatur cara kerja halaman tersebut.

Berbeda dengan HTML dan CSS yang hanya bersifat penampilan, JavaScript adalah bahasa pemrograman sejati yang memiliki logika, perhitungan, pengambilan keputusan, dan pengulangan. Saat ini JS tidak hanya berjalan di peramban saja, tapi juga bisa dipakai di sisi server dengan teknologi seperti Node.js, bahkan untuk membuat aplikasi seluler dan perangkat lunak desktop.

Sejarah Singkat JavaScript

JavaScript pertama kali dibuat oleh Brendan Eich pada tahun 1995 saat bekerja di perusahaan Netscape. Awalnya bernama "Mocha", kemudian diubah menjadi "LiveScript", dan akhirnya menjadi JavaScript untuk memanfaatkan popularitas bahasa Java saat itu — meskipun keduanya tidak memiliki hubungan khusus.

  • 1997: JS distandarisasi menjadi spesifikasi bernama ECMAScript.
  • 2009: ECMAScript 5 dirilis, membawa banyak perbaikan fitur.
  • 2015: ECMAScript 6 (atau ES6/ES2015) muncul, menjadi tonggak utama dengan fitur yang jauh lebih lengkap dan modern.
  • Sekarang: Terus dikembangkan setiap tahun dengan fitur baru yang semakin memudahkan pengembangan.

Cara Menyisipkan JavaScript ke Halaman Web

Sama seperti CSS, ada 3 cara utama memasukkan kode JS ke dalam dokumen HTML:

1. JS Inline

Ditulis langsung di dalam atribut elemen HTML, biasanya untuk aksi sederhana.

<button onclick="alert('Halo Dunia!')">Klik Saya</button>

2. JS Internal

Ditulis di dalam tag <script> yang bisa diletakkan di bagian <head> atau lebih disarankan di bagian akhir <body> agar halaman lebih cepat dimuat.

<body>
  <h1>Judul Halaman</h1>

  <script>
    console.log("Kode JS berjalan dengan baik!");
  </script>
</body>

3. JS Eksternal

Kode ditulis di berkas terpisah dengan ekstensi .js, lalu dipanggil lewat tag <script>. Cara ini paling disarankan agar kode rapi dan bisa dipakai di banyak halaman.

<!-- Di dalam berkas HTML -->
<script src="kode_saya.js"></script>

Isi berkas kode_saya.js:

// Ini adalah komentar satu baris
let pesan = "Selamat Belajar JavaScript!";
console.log(pesan);

Variabel dan Tipe Data

Variabel adalah tempat menyimpan nilai atau data agar bisa dipakai dan diubah nanti.

Cara Membuat Variabel

Ada 3 kata kunci yang sering dipakai:

  • var: Cara lama, masih berfungsi tapi jarang dipakai sekarang.
  • let: Untuk nilai yang bisa diubah isinya.
  • const: Untuk nilai yang tetap dan tidak bisa diubah lagi setelah ditetapkan.
let nama = "Andi";
const umur = 25;
let sudahMenikah = false;

nama = "Budi"; // Bisa diubah
// umur = 30; → Ini akan menimbulkan error karena const tidak bisa diubah

Jenis Tipe Data Utama

  • String: Teks atau tulisan, ditulis di dalam tanda kutip ' ', " ", atau ` `.
  • Number: Angka, bisa berupa bilangan bulat atau desimal.
  • Boolean: Nilai logika, hanya ada dua: true (benar) dan false (salah).
  • Array: Kumpulan banyak data dalam satu variabel, ditulis dengan tanda kurung siku [ ].
  • Object: Kumpulan data berpasangan antara kunci dan nilai, ditulis dengan tanda kurung kurawal { }.
  • Undefined: Variabel yang sudah dibuat tapi belum diberi nilai.
  • Null: Menyatakan nilai kosong atau tidak ada isi.

Operator dalam JavaScript

Operator berfungsi melakukan operasi pada nilai atau variabel:

🔹 Operator Aritmatika

let a = 10, b = 3;
console.log(a + b); // Penjumlahan → 13
console.log(a - b); // Pengurangan → 7
console.log(a * b); // Perkalian → 30
console.log(a / b); // Pembagian → 3.333...
console.log(a % b); // Sisa pembagian → 1
console.log(a ** b); // Pangkat → 1000

🔹 Operator Perbandingan

Menghasilkan nilai benar atau salah:

  • == → Sama nilai saja
  • === → Sama nilai dan tipe data (lebih disarankan)
  • != → Tidak sama nilai
  • !== → Tidak sama nilai atau tipe data
  • > → Lebih besar
  • < → Lebih kecil
  • >= → Lebih besar atau sama
  • <= → Lebih kecil atau sama

🔹 Operator Logika

  • && → DAN, kedua syarat harus benar
  • || → ATAU, cukup salah satu syarat benar
  • ! → BUKAN, membalikkan nilai logika

Percabangan: Mengambil Keputusan

Percabangan dipakai untuk menjalankan perintah tertentu jika syarat terpenuhi.

Struktur if...else

let nilai = 80;

if (nilai >= 75) {
  console.log("Lulus!");
} else {
  console.log("Belum Lulus");
}

Struktur if...else if...else

let nilai = 85;

if (nilai >= 90) {
  console.log("Nilai A");
} else if (nilai >= 80) {
  console.log("Nilai B");
} else if (nilai >= 70) {
  console.log("Nilai C");
} else {
  console.log("Nilai D");
}

Perulangan: Mengulang Perintah

Digunakan untuk menjalankan perintah yang sama secara berulang tanpa menulis kode berulang-ulang.

🔹 Perulangan for

Cocok jika jumlah pengulangan sudah diketahui pasti:

for (let i = 1; i <= 5; i++) {
  console.log("Pengulangan ke-" + i);
}

🔹 Perulangan while

Akan berjalan selama syarat bernilai benar:

let hitung = 1;
while (hitung <= 3) {
  console.log("Hitung: " + hitung);
  hitung++;
}

Fungsi: Kode yang Bisa Dipakai Ulang

Fungsi adalah sekumpulan kode yang diberi nama dan bisa dipanggil kapan saja.

// Membuat fungsi
function sapa(namaPengguna) {
  return "Halo, " + namaPengguna + "!";
}

// Memanggil fungsi
let ucapan = sapa("Voxz");
console.log(ucapan); // Hasil: Halo, Voxz!

Mengenal DOM: Menghubungkan JS dengan HTML

DOM adalah singkatan dari Document Object Model. Ini adalah cara JavaScript mengakses, mengubah, menambah, atau menghapus elemen dan isi dari halaman HTML.

Contoh cara mengambil elemen:

// Ambil berdasarkan ID
const judul = document.getElementById("judul-utama");

// Ambil berdasarkan kelas
const kotak = document.getElementsByClassName("kotak");

// Ambil dengan selektor CSS
const paragraf = document.querySelector("p");

// Mengubah isi teks
judul.textContent = "Judul Baru yang Diubah Lewat JS";

// Mengubah gaya CSS
judul.style.color = "#3B82F6";

Menangani Peristiwa (Event)

JS bisa merespons tindakan pengguna seperti mengklik, mengetik, menggeser layar, dll.

const tombol = document.getElementById("tombol-aksi");

tombol.addEventListener("click", function() {
  alert("Tombol berhasil diklik!");
});
💡 Catatan Penting: JavaScript adalah bahasa yang sangat fleksibel dan luas. Setelah menguasai dasar-dasar ini, kamu bisa melanjutkan ke materi lanjutan seperti array, objek, manipulasi data, pengambilan data dari internet, hingga membuat aplikasi lengkap. Kuncinya adalah banyak mencoba dan memecahkan masalah lewat kode.

Kesimpulan

JavaScript adalah kunci untuk membuat halaman web menjadi hidup dan bisa berinteraksi. Bagi kamu yang ingin membuat bot, alat bantu, atau situs web yang fungsional, JS adalah bahasa yang wajib dikuasai. Awalnya mungkin terasa banyak istilah baru, tapi dengan latihan rutin dan menerapkannya ke proyek nyata, kamu akan semakin terbiasa dan mahir menggunakannya.
