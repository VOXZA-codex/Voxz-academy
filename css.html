🎨 PEMBELAJARAN LENGKAP CSS

Apa Itu CSS?

CSS adalah singkatan dari Cascading Style Sheets. Ini adalah bahasa yang digunakan untuk mengatur tampilan, desain, dan tata letak halaman web yang sudah dibuat dengan HTML. Jika HTML berfungsi sebagai "kerangka" atau struktur bangunan, maka CSS berperan sebagai "cat, warna, bentuk, dan tata ruang" yang membuat bangunan terlihat menarik dan rapi.

Tanpa CSS, halaman web hanya akan terlihat sebagai teks biasa dengan tulisan hitam di atas latar putih, tanpa warna, tata letak, atau efek apapun. Dengan CSS, kamu bisa mengatur hampir semua aspek tampilan: warna, ukuran huruf, jarak antar elemen, posisi, bentuk sudut, bayangan, animasi, dan banyak lagi.

Sejarah Singkat CSS

CSS pertama kali diusulkan pada tahun 1994 oleh Håkon Wium Lie untuk mengatasi masalah pengaturan tampilan yang rumit saat itu. Sebelum ada CSS, pengaturan tampilan harus ditulis langsung di dalam kode HTML, sehingga kode menjadi panjang, berantakan, dan sulit diubah.

  • CSS 1: Dirilis tahun 1996, mendukung pengaturan warna, huruf, jarak, dan perataan teks.
  • CSS 2: Keluar tahun 1998, menambahkan fitur tata letak, posisi elemen, dan media cetak.
  • CSS 2.1: Menjadi standar yang paling stabil dan banyak dipakai selama bertahun-tahun.
  • CSS 3: Versi terbaru yang dikembangkan secara bertahap, mendukung efek bayangan, sudut lengkung, gradasi warna, animasi, tata letak fleksibel, dan kompatibilitas untuk perangkat seluler.

Cara Menyisipkan CSS ke HTML

Ada 3 cara utama untuk menghubungkan kode CSS dengan dokumen HTML:

1. CSS Inline

Ditulis langsung di dalam tag HTML menggunakan atribut style. Cara ini hanya memengaruhi satu elemen saja.

<p style="color: blue; font-size: 18px;">Teks ini berwarna biru dan ukurannya 18 piksel</p>

2. CSS Internal

Ditulis di dalam bagian <head> menggunakan tag <style>. Berlaku untuk seluruh elemen dalam satu halaman saja.

<head>
  <style>
    p {
      color: #333;
      line-height: 1.6;
    }
  </style>
</head>

3. CSS Eksternal

Ditulis di berkas terpisah dengan ekstensi .css, lalu dipanggil di dalam HTML. Cara ini paling disarankan karena bisa dipakai di banyak halaman sekaligus dan kode menjadi lebih rapi.

Di dalam HTML:

<head>
  <link rel="stylesheet" href="gaya.css">
</head>

Di dalam berkas gaya.css:

body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
}

h1 {
  color: #2c3e50;
  text-align: center;
}

Sintaks Dasar CSS

Setiap aturan CSS memiliki pola penulisan yang tetap:

selektor {
  properti: nilai;
  properti_lain: nilai_lain;
}
  • Selektor: Menentukan elemen mana yang akan diubah tampilannya.
  • Properti: Jenis pengaturan yang ingin diterapkan (misal: warna, ukuran, jarak).
  • Nilai: Pengaturan spesifik yang diberikan pada properti tersebut.
  • Titik koma (;): Menandai akhir dari satu baris perintah.

Jenis-Jenis Selektor CSS

Selektor adalah kunci untuk memilih elemen yang akan diatur. Berikut yang paling sering dipakai:

🔹 Selektor Elemen

Memilih semua elemen dengan nama tag yang sama.

p {
  color: #333;
}

🔹 Selektor Kelas

Memilih elemen berdasarkan atribut class. Ditandai dengan tanda titik (.). Bisa dipakai berulang kali.

/* Di CSS */
.kotak {
  background: #ecf0f1;
  padding: 15px;
}

/* Di HTML */
<div class="kotak">Isi kotak</div>

🔹 Selektor ID

Memilih elemen berdasarkan atribut id. Ditandai dengan tanda pagar (#). Hanya boleh dipakai satu kali per halaman.

/* Di CSS */
#judul-utama {
  font-size: 26px;
  color: #2980b9;
}

/* Di HTML */
<h1 id="judul-utama">Judul Halaman</h1>

🔹 Selektor Universal

Menerapkan aturan ke semua elemen di halaman. Ditandai dengan tanda bintang (*).

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

Properti CSS yang Sering Digunakan

🔹 Mengatur Teks dan Huruf

  • color → Mengatur warna teks.
  • font-family → Menentukan jenis huruf.
  • font-size → Mengatur ukuran huruf.
  • font-weight → Mengatur ketebalan huruf.
  • text-align → Meratakan teks (kiri, tengah, kanan, rata).
  • text-decoration → Memberi garis bawah, atas, atau mencoret teks.
  • line-height → Mengatur jarak antar baris teks.
  • letter-spacing → Mengatur jarak antar huruf.

🔹 Mengatur Warna dan Latar

  • background-color → Warna latar belakang elemen.
  • background-image → Menampilkan gambar sebagai latar.
  • background-repeat → Mengatur apakah gambar diulang atau tidak.
  • background-size → Mengatur ukuran gambar latar.
  • opacity → Mengatur tingkat transparansi elemen (nilai 0 sampai 1).

🔹 Mengatur Ukuran dan Jarak

Setiap elemen memiliki ruang di luar dan di dalamnya, dikenal sebagai model kotak:

  • width → Lebar elemen.
  • height → Tinggi elemen.
  • margin → Jarak di luar batas elemen.
  • padding → Jarak di dalam batas elemen.
  • border → Garis tepi yang membatasi elemen.
  • border-radius → Membuat sudut elemen menjadi melengkung.

🔹 Mengatur Posisi dan Tata Letak

  • position → Mengatur cara penempatan elemen: static, relative, absolute, fixed, sticky.
  • display → Mengatur cara elemen ditampilkan: block, inline, inline-block, flex, grid.
  • float → Mengatur posisi elemen mengapung ke kiri atau kanan.
  • z-index → Mengatur urutan tumpukan elemen (mana yang terlihat di atas).

🔹 Efek Tambahan

  • box-shadow → Memberikan bayangan pada elemen.
  • text-shadow → Memberikan bayangan pada teks.
  • transition → Membuat perubahan tampilan menjadi halus dan bertahap.
  • transform → Memutar, memperbesar, memperkecil, atau memiringkan elemen.

Satuan Ukuran dalam CSS

CSS memiliki beberapa jenis satuan ukuran, dibagi menjadi dua kelompok:

✅ Satuan Mutlak

Ukuran tetap yang tidak berubah mengikuti kondisi layar:

  • px → Piksel, paling sering dipakai.
  • cm → Sentimeter.
  • mm → Milimeter.
  • in → Inci.

✅ Satuan Relatif

Ukuran yang mengikuti ukuran elemen induk atau layar, sangat disarankan untuk desain yang bisa menyesuaikan perangkat:

  • % → Persen, mengikuti ukuran elemen induk.
  • em → Mengikuti ukuran huruf elemen induk.
  • rem → Mengikuti ukuran huruf dasar halaman.
  • vw → Persen dari lebar jendela layar.
  • vh → Persen dari tinggi jendela layar.
💡 Catatan Penting: CSS bekerja dengan aturan "mengalir dan menumpuk". Jika ada lebih dari satu aturan yang memengaruhi elemen yang sama, aturan yang paling spesifik atau yang ditulis paling akhir akan berlaku. Ini disebut prinsip Cascading.

Kesimpulan

Menguasai CSS adalah langkah kedua yang sangat penting setelah memahami HTML. Dengan CSS, kamu bisa mengubah tampilan halaman sederhana menjadi desain yang menarik, rapi, dan nyaman dilihat. Kuncinya adalah sering mencoba, mengubah nilai-nilai yang ada, dan melihat hasilnya langsung di browser. Semakin sering dipraktikkan, semakin cepat kamu menguasai cara mengatur tampilan sesuai keinginan.
